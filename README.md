# Emmanuel Tigoue — Security Engineer Portfolio

[![Live Site](https://img.shields.io/badge/Site-Live-00FF41?style=flat-square)](https://et-sec.github.io/portfolio/)

Interactive portfolio showcasing security engineering work, certifications, and technical capabilities.

## Features

- Dark/military HUD aesthetic with matrix rain canvas
- Skill radar chart with animated fill
- Terminal Easter egg (triple-click the logo)
- Keyboard navigation (J/K/T/R/?)
- Konami code Easter egg
- Responsive design with 19 CSS animations
- Schema.org structured data for SEO

## Stack

Single `index.html` — all CSS and JS inline. Zero dependencies. Zero build step.
